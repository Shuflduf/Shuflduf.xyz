# Features
- TWO WORLDS, LIGHT AND DARK
  - (There are two ways to view this website)
- **Switch between the Light and Dark worlds by finding the portal buttons**
  - In the Light world, this is accessed by pulling the sidebar out with the mouse
  - In the Dark world, its accessed by pulling down the "Personal" header in the navigation links
- [Ukiyo](https://github.com/technerium/obsidian-ukiyo) theme for Light world
- Indie Web styles for the Dark world (from [Neocities](https://neocities.org/))
- Full mobile support (except for some games)
